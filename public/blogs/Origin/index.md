# 安装步骤

0.在中国移动云盘电脑备份文件夹中下载

1.选中下载的【Origin 2022(64bit)】软件压缩包，鼠标右击（win11系统先点击“显示更多选项”）选择【解压到Origin 2022(64bit)】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/6b4b80c1-c2f9-4f0a-9522-802b1e7e8d71.png)

2.双击打开解压后的【Origin 2022(64bit)】文件夹。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/943fcaf9-7528-4286-8088-757e7d0cc9b2.png)

3.双击打开【Setup】文件夹。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/60d59caa-eb89-48df-aed9-b6569dbb4c6b.png)

4.鼠标右击【Setup】选择【以管理员身份运行】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/1f67f765-5d2a-4c8c-8ceb-bc1dfb9177bf.png)

5.点击【下一步】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/16ac5c9c-d2b9-4777-b088-9bf6a10a2874.png)

6.选择【我接受许可证协议中的条款】，点击【下一步】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/20b337c5-5d71-49b3-812b-c0659d1a5fae.png)

7.选择【安装OriginPro试用版】，点击【下一步】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/25a4d362-8882-4727-8de1-3660df5168d8.png)

温馨提示：部分用户安装的时候提示“试用期已结束”请查看解决方法。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/94fc3864-70cb-49ed-abc0-f786563b7376.png)

解决办法：将电脑时间调整为2019年2月17日后再安装，安装完成后再调回来。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/67d41249-fb37-487a-a08c-696ff442ac6c.png)

8.点击【确定】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/7433db8a-734b-4369-b5a1-d39c248d966b.png)

9.用户名和公司名称随意填写，然后点击【下一步】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/c6937969-559b-402e-b60d-b2a0565682ae.png)

10.点击【是】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/6e37a228-b14f-463e-b490-34fedb98149e.png)

11.修改路径地址中的首字符C可更改安装位置（建议安装除C盘以外的其他磁盘），点击【下一步】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/645ef5b4-7739-4107-9cb2-da7403e94bfb.png)

12.点击【是】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/1c89e161-6d55-48a6-b075-e0cc71b8feb2.png)

13.点击【下一步】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/cecf620f-eb82-4bc9-ae43-8519787cfbc6.png)

14.点击【下一步】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/2cf4b22b-9a6f-4e2d-95cd-b8d6c1c4487b.png)

15.点击【下一步】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/ae5b525d-9ae0-4055-9a7a-7adf79a5c832.png)

16.点击【下一步】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/923fe6dc-3973-42de-8cb6-7332236ff031.png)

17.软件正在安装中。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/12d5bc4c-3cd4-4e9e-bcc6-70c86e9f58ea.png)

18.取消勾选【打开Readme.txt】，点击【完成】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/208f1c0a-7edb-4112-a440-7ca547a19cf3.png)

19.打开安装包解压后的【Origin 2022(64bit)】文件夹，双击打开【Crack】文件夹。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/876ee879-c50f-4ddd-ba92-9f5d1bba874b.png)

20.鼠标右击【ok.dll】选择【复制】。

温馨提示：如果文件夹里面没有【ok.dll】文件，请关闭所有杀毒软件、Win10/11系统还需关闭“Windows安全中心实时保护”后重新解压。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/8a972ac1-af3e-4595-b949-8cdf2a651b49.png)

21.创建桌面启动快捷方式：点击桌面左下角【开始图标】，（win11点击"所有应用"）在菜单栏中找到并拖动【Origin 2022】图标到电脑桌面。。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/bdcab137-53c6-40df-bf00-d8cda6ac7d06.png)

22.在桌面鼠标右击【Origin 2022】选择【打开文件所在的位置】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/4c25fef1-c457-4925-ae16-91129076e431.png)

23.在空白处鼠标右击选择【粘贴】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/483ed81b-121a-4bc3-b2c1-37934448f0d9.png)

24.点击【替换目标中的文件】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/bcf6d8c8-c7cc-4106-87bc-a7b30608bc52.png)

25.打开安装包解压后的【Origin 2022(64bit)】文件夹，双击【Origin 2022切换中文】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/404b28a8-57bf-438a-bb1e-fa663dd35fe7.png)

26.点击【是】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/5e728778-c390-4fc9-aa1e-619276a8d22c.png)

27.点击【确定】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/ba4b3620-ed6b-4435-8d16-4b090b2d032b.png)

28.在桌面双击【Origin 2022】图标启动软件。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/00e37704-e775-497f-8711-154cb7ac3605.png)

29.点击【确定】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/b1dfa52d-d25c-4b1c-9642-9809c921cc5e.png)

30.点击【是】。

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/32266117-4086-46b7-93a2-9839a4b49bdb.png)

31.软件安装成功！

![](https://s.xinzhi-cdn.zone/user/6926a30041c8233286f21192/695fe9d876d08558d43eabac/91c79fe9-4b70-4728-98ce-8c2342abb092.png)
